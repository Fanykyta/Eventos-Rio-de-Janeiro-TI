A API organiza os eventos da área de tecnologia na região do Rio de Janeiro com o intuito de formar um calendário que informe quando e onde está ocorrendo e se há disponibilidade de vagas.

A API recebe dados através de cadastro dos eventos por parte dos organizadores.

Tem como objetivo centralizar os eventos de tecnologias em uma mesma plataforma de fácil acesso e ainda facilitar a divulgação dos mesmos pelas redes sociais e através do hotsite.
